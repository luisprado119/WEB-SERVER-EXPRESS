Nuevo archivo estático

http://localhost:3333/myIndex.html

Este archivo HTML es servido directamente desde la carpeta /public gracias a express.static(), sin necesidad de definir una ruta específica.

¿Cómo llega este fichero al navegador? El servidor tiene configurado app.use(express.static('public')), que monta toda la carpeta /public bajo la URL raíz. Cualquier fichero dentro de /public es accesible directamente por su nombre.

⚙️ Cómo funciona express.static()

📂 Configuración básica

Una sola línea de código sirve todos los ficheros de una carpeta:

// Sirve la carpeta /public bajo la URL raíz "/"
app.use(express.static('public'))

// Acceso en el navegador:
//  public/index.html   → http://localhost:3333/
//  public/myIndex.html → http://localhost:3333/myIndex.html
//  public/guia.html    → http://localhost:3333/guia.html

🗂️ Segunda carpeta con prefijo URL

Puedes tener múltiples carpetas estáticas con diferentes prefijos de URL:

// Sirve la carpeta /docs bajo la URL "/docs"
app.use('/docs', express.static('docs'))

// Acceso: http://localhost:3333/docs/index.html
