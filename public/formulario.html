Formularios de Práctica Express

Practica el envío de datos al servidor vía POST: JSON, form-urlencoded y multipart. Cada sección muestra la respuesta en tiempo real.

POST Enviar datos JSON /data

Envía un objeto JSON al servidor. Express lo parsea con express.json() y lo devuelve con status 201 Created. Los datos llegan en req.body.

POST /data 

      

    

  


  
  

    

      POST
      Echo del body (form-urlencoded)
      /echoPost
    

    

      Envía un formulario clásico application/x-www-form-urlencoded.
      Llega en req.body gracias a express.urlencoded().
      El servidor devuelve exactamente lo que recibió.

      POST
      JSON + Query String combinados
      /echoPostJson?foo=bar
    

    

      Demuestra que una petición puede tener datos en el body (JSON) y en la query string (?foo=bar) al mismo tiempo.
      Express los expone por separado: req.body y req.query.
    

    

      

        

          
          
        

        

          
          
        

        

          
          
        

      

      
      

        

          
          
          POST /echoPostJson
          
        

        

      

    

  


  
  

    

      POST
      Params de ruta + Query + Body
      /echoParamsPost/:cliente/facturas/:factura
    

    

      Las tres fuentes de datos de una petición Express: req.params (ruta dinámica), req.query (query string) y req.body (JSON). El servidor devuelve los tres juntos.
    

    

      

        Ejemplo de URL generada:

        /echoParamsPost/ACME/facturas/F2024001?formato=json
      

      

        

          
          
        

        

          
          
        

        

          
          
        

        

          
          
        

      

      
      

        

          
          
          
          
        

        

      

    

  


  
  

    

      POST
      Subida de ficheros genérica
      /uploadFicheros
    

    

      Envía ficheros usando enctype="multipart/form-data". El middleware express-fileupload los intercepta y los pone en req.files. Puedes adjuntar también datos de texto con los ficheros.
