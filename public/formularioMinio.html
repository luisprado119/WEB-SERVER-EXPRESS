🗄️ MinIO Object Storage

Gestiona buckets y ficheros directamente desde el navegador

API → http://localhost:9000  |  Express → http://localhost:3333

Gestión de Buckets

📦

Listar Buckets

GET /minio/buckets

Devuelve todos los buckets existentes con su nombre y fecha de creación.

Pulsa el botón para ver los buckets…
➕

Crear Bucket

POST /minio/buckets/:bucket

El nombre debe estar en minúsculas, sin espacios ni caracteres especiales.

Gestión de Ficheros

🗂️

Listar Ficheros

GET /minio/buckets/:bucket/files
Selecciona un bucket y pulsa el botón…
⬆️

Subir Fichero

POST /minio/buckets/:bucket/upload
📁
Arrastra un fichero aquí o haz clic para seleccionarlo
Selecciona un fichero para subir…

Recuperar y Eliminar Ficheros

⬇️

Recuperar Fichero

GET /minio/buckets/:bucket/files/:filename

Descarga el fichero a través de Express usando streaming. El fichero pasa por el servidor (a diferencia de la URL firmada).

Rellena los campos y descarga…
🗑️

Eliminar Fichero

DELETE /minio/buckets/:bucket/files/:filename

Compara los dos patrones de escritura de código asíncrono en Node.js.

Promise chaining (/v1)
removeObject().then(ok).catch(err)

async / await
await removeObject() · try/catch

URL Firmada (Presigned URL)

🔗

Generar URL de Descarga Firmada

GET /minio/buckets/:bucket/files/:filename/url

La URL firmada permite descargar el fichero directamente desde MinIO sin pasar por Express, durante el tiempo indicado. Incluye una firma criptográfica que expira.

Rellena los campos y pulsa "Generar URL firmada"…
